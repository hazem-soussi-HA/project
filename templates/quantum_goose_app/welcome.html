Django Backend

The Django backend provides a robust foundation with:

  • Templates: Organized in templates/quantum_goose_app/
  • Static Files: CSS, JS, and images in static/
  • API Endpoints: RESTful APIs for React integration
  • Database: SQLite with Django ORM
  • Admin Panel: Available at /admin/
Available Endpoints:
  • GET / - Welcome page
  • GET /admin/ - Django admin
  • GET /quantum-goose-app/api/ - API info
  • GET /quantum-goose-app/health/ - Health check

React Frontend

The React frontend application is located in quantum-goose-app/:

  • Source: quantum-goose-app/src/
  • Build: quantum-goose-app/dist/
  • Public Assets: quantum-goose-app/public/
  • Package Manager: npm/pnpm
React App Integration:

Quick Start Guide

1. Run the Django Development Server:
cd /d/project
python manage.py runserver
2. Build the React App (if needed):
cd /d/project/quantum-goose-app
npm run build
3. Access the Applications:
  • Django Welcome: http://localhost:8000/
  • React App: http://localhost:8000/react/
  • Django Admin: http://localhost:8000/admin/
  • API Documentation: http://localhost:8000/quantum-goose-app/api/
4. Test the Integration:
curl http://localhost:8000/quantum-goose-app/health/
